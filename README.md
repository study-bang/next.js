1. React Foundations
- `npm install react@latest react-dom@latest next@latest`

2. App Router
- `npm install -g pnpm`
- `npx create-next-app@latest nextjs-dashboard --example "https://github.com/vercel/next-learn/tree/main/dashboard/starter-example" --use-pnpm`
- `cd nextjs-dashboard`
- `pnpm i`
- `pnpm dev`
